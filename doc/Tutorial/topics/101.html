3.5.2 Using foreign key

A foreign key field is treated as any other field during querying or presenting data of a table sequence. But you need to pay attention to the data type it uses.

For example, to select from the CITIES table the cities that are located in the states whose names contain ¡°la¡±, you can directly call the NAME field of the foreign key State in the filtering expression:
