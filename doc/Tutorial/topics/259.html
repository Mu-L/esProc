Chapter11 Composite Tables

You can choose to skip this section if you are not a professional programmer. This won’t affect your learning about the other contents of this Tutorial.

In esProc data warehouse, a composite table is a table consisting of multiple entity tables having their own structures; it has multilevel primary keys. In an unambiguous context, an entity table can be just called the table.
