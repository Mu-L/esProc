Main Path

When the main directory is not specified, there are three types of cases for executing a cellset file in esProc designer:

If the current .splx/.dfx file is saved, the main directory is the directory where the file exists;

If the current .splx/.dfx file isn¡¯t saved, the main directory is the directory where the last used file is stored;
