Start Office Add-in Service

This is an esProc Desktop edition feature that can run on both Windows and Mac.

By checking this item, you can use SPL.X() function to execute the SPL script, get the result set, and populate it to the Excel.

Installations and configurations of the Office add-in service for Windows and Mac are different. Details are explained in Office Add-in Service Installation and Configuration.
