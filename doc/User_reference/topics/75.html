Search path

It specifies the directory for storing program files. There can be multiple Search paths, separated by semicolons.

A Search path can be a relative path or an absolute path. For a relative path, the relative directory on IDE is [installation directory]/esProc, and that on WEB side is the application root path.

When searching for a desired file in a cellset program using file@s, search files in non-absolute paths according to the specified order – class path -> search path -> main directory.
