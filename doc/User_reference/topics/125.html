RSRV

This is an esPoc Enterprise edition feature.

You can find the RSRV functionality on the menu bar in esProc designer. The functionality is used to log on, connect to and manage files on RSRV. The file management operations include upload, download, delete and script debugging.

Learn more about the deployment and uses of RSRV in Deploy and Use RSRV Enterprise Edition.
