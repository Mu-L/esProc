asin()

Description:

Compute arc sine value.

Syntax:

asin(number)

Note:

The function computes arc sine value of a real number.

Parameter:

number

A real number between -1 and 1.

Return value:

Numeric value

Example:

asin(-1)

-1.5707963267948966

asin(sin(pi()/2))

1.5707963267948966

asin(sin(0))

0.0

Related functions:

acos()

atan()
