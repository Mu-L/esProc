rands()

Description:

Get a random string.

Syntax:

rands(s,l)

Note:

The function generates a string of length l randomly using the characters of string s.

Parameter:

s

A string.

l

An integer.

Return value:

A string

Example:

rands("abc",5)

Generate a string of length 5 with characters of string ¡°abc¡± .
