Modulus

Description:

Perform Mod operation on two integers or real numbers; the result can be long integers and real numbers.

 

Perform integer division on two decimals without the factional parts or two long integers.

Syntax:

x%y

Get remainder.

x\y

Get the integer part.

Parameter:

x

A long integer or a real number.

y

A long integer or a real number.

Return value:

Integer/Real number

Example:

=7%2

1
