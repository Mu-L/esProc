spark_close()

Description:

Close a Spark database connection.

Syntax:

spark_close(con)

Note:

This external library function (See External Library Guide) closes connection to a Spark database.

Parameter:

con

Spark database connection

Example:

 

A

 

1

=spark_open("hdfs://192.168.0.8:9000","thrift://192.168.0.8:9083","aa")

 

2

=spark_close(A1)

Close connection to the spark database.
