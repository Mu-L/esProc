P.o()

Description:

Convert a pure table sequence to an ordinary sequence.

Syntax:

P.o()

Note:

The function converts pure table sequence P to an ordinary sequence. Only supported by esProc Enterprise Edition.

Parameter:

P

A pure table sequence.

Return value:

Table sequence
