$(d)dql;...

Description:

Execute DQL query statement.

Syntax:

$(d)dql;…

Parameter:

d

DQL connection.

dql

DQL query statement.

 

…

Arguments in the DQL qiery.

 

Return value:

Table sequence

Example:

 

A

 

1

=dql("demo.glmd")

Connect to demo.glmd in the main directory.

2

$(A1)SELECT OrderID,Amount FROM Orders

Execute the DQL query statement.

3

$(dql("demo.glmd"))SELECT OrderID,Amount FROM Orders

Same as above.

4

$(A1)SELECT  OrderID,Amount FROM Orders where  OrderID=?;10353

Execute the DQL query that has an argument.
