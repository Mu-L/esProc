r2dbc_close()

Description:

Close R2DBC connection.

Syntax:

r2dbc_close(r2dbc_open)

Note:

The external library function (See External Library Guide) closes R2DBC connection.

Parameter:

r2dbc_open

R2DBC connection.

Example:

 

A

 

1

=r2dbc_open("r2dbcs:mysql://root:123456@127.0.0.1:3306/mytest")

Connect to mysql database through r2dbc.

2

=r2dbc_close(A1)

Close r2dbc connection.
