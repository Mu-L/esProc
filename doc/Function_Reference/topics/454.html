hive_db()

Description:

Obtain a list of Hive databases.

Syntax:

hive_db(con)

Note:

External library function (See External Library Guide for uses).

 

It obtains a list of Hive databases.

Parameter:

con

Database connection string.

Return value:

Table sequence

Example:

 

A

 

1

=hive_open("hdfs://192.168.0.8:9000","thrift://192.168.0.8:9083","hive","asus")

Connect to Hive database.

2

=hive_db(A1)

Return a table sequence.

3

=hive_close(A1)
