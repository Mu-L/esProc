acos()

Description:

Compute arc cosine value.

Syntax:

acos(number)

Note:

The function computes arc consine value of a real number between -1 and 1.

Parameter:

number

A real number between -1 and 1e.

Return value:

Numeric value

Example:

acos(-1)

3.141592653589793

acos(cos(pi()/2)) 

1.5707963267948966

acos(cos(0))

0.0
