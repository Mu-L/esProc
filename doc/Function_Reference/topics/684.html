olap_close()

Description:

Close connection to a multidimensional database.

Syntax:

olap_close(conn)

Note:

This external library function (See External Library Guide) closes the connection to a multidimensional database.

Parameter:

conn

Database connection string.

Example:

 

A

 

1

=olap_open("http://192.168.0.178:8088/msmdpump.dll","CubeTest","Administrator","admin")

 

2

>olap_close(A1)

Close connection to the multidimensional database.
