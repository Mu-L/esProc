G.hlink

Description:

Return the hyperlink string of a canvas.

Syntax:

G.hlink()

Note:

The function gets a hyperlink string corresponding to the returned non-SVG image.

Return value:

Hyperlink string

Example:

a.hlink()

Return a hyperlink string.
