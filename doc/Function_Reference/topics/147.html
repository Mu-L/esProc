channel(cs)

Description:

Create a channel and push data in a specified cursor into it.

Syntax:

channel(cs)

Note:

The function creates a channel, pushes data in cursor cs into it, and returns a channel. This is equivalent to cs.push(channel()).

Parameter:

cs

A cursor.

Return value:

Channel
