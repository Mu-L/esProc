f.open()

Description:

Generate a cluster table from a cluster file.

Syntax:

f.open(p)

Note:

Then function opens cluster file f and generates a cluster able based on f.

Parameter:

f

A cluster file; support only the composite table file.

Return value:

Cluster table

Example:

 

A

 

1

=file("employee.ctx",["192.168.0.104:8281"])

Open a cluster file.

2

=A1.open()

Generate a cluster table based on it.
