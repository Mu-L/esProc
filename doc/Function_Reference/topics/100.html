atan()

Description:

Compute arc tangent value of the specified data.

Syntax:

atan (number)

Note:

The parameter number is a real number.

Parameter:

number

The real number for which you want to compute the arc tangent.

Return value:

Arc tangent

Example:

atan(1)

0.7853981633974483

atan(tan(pi()/2))

1.5707963267948966

atan(tan(0))

0.0

asin()
