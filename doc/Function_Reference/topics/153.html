chi2inv()

Description:

A chi-square inverse cumulative distribution function.

Syntax:

chi2inv(p, v)

Note:

The external library function (See External Library Guide) is a chi-square inverse cumulative distribution function (ICDF).

Parameter:

P

The p-value within the interval (0,1) .

v

Degrees of freedom (DF) .

Return value:

Double-precision number

Example:

 

A

 

1

=chi2inv(0.95,10)
