P.create()

Description:

Generate a new empty table sequence by copying data structure of a specific record sequence.

Syntax:

P.create()

Note:

If parameter P has the key, just copy it.

Parameter:

P

A record sequence.

Return value:

Empty table sequence

Example:

Generated from a record sequence:
