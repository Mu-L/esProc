ifnumber()

Description:

Find whether the specified data is numeric type.

Syntax:

ifnumber(Exp)

Note:

The function finds whether the parameter Exp is of numeric data type.

Parameter:

Exp

An expression of any data type

Return value:

Boolean

Example:

ifnumber("abc")

false

ifnumber("1234")

false

ifnumber(1234)

true

ifnumber("1234sss")

false
