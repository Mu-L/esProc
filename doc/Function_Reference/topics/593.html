ln()

Description:

Compute the natural logarithm of the specified data.

Syntax:

ln(numberExp)

Note:

The function computes the natural logarithm of parameter numberExp.

Parameter:

numberExp

Data for which you want to compute the natural logarithm.

Return value:

Numeric value

Example:

ln(54)

3.9889840465642745

Related function:

lg()
