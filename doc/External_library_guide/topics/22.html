3.19        Oss

1. The directory containing files of this external library is: installation directory\esProc\extlib\OssCli. The Raqsoft core jar for this external library is scu-ali-oss-cli-2.10.jar.

aliyun-java-sdk-core-4.5.10.jar

aliyun-java-sdk-kms-2.11.0.jar

aliyun-java-sdk-ram-3.1.0.jar

aliyun-sdk-oss-3.17.3.jar

jdom2-2.0.6.1.jar

jettison-1.5.4.jar

Note: The third-party jars are encapsulated in the compression package and users can choose appropriate ones for specific scenarios.

 

2. OssCli requires a JRE version 1.7 or above. Users need to install a higher version if the esProc built-in JRE version does not meet the requirements, and then configure java_home in config.txt under installation directory \esProc\bin. Just skip this step if the JRE version is adequate.

 

3. esProc provides a series functions – oss_bucket(), oss_close(), oss_copy(), oss_file(), oss_list() and oss_open() to access Oss cloud. Look them up in【Help】-【Function reference】to find their uses.
